# ISRO_App
It is project to show case ISRO(INDIAN SPACE RESEARCH ORGANISATION) launches and spacecraft information.

# Tech Stack:
- Kotlin
- Android studio
- Jetpack Compose
- Retrofit
- MVVM Architecture
- Navigation component
- BottomNavBar
